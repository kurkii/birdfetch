# birdfetch
system information fetcher for birds made in Go (only works on linux)

# how to coompile
requirements: Go
1. download sauce code
2. `cd birdfetch`
3. `go build`
4. `sudo cp birdfetch /usr/bin`
5. `birdfetch`

or use my precompiled binary if you trust me enough
